# 附录 A：常见问题速查

## 安装相关

**Q: 安装报错"command not found: curl"**
A: 先装 curl。Mac 通常自带，如果没有：在终端输入 `xcode-select --install`，按提示完成。Windows 去 https://curl.se 下载安装。

**Q: 安装过程中提示输入密码**
A: 输入你电脑的开机密码。输入时屏幕不会显示任何字符，这是正常的。输完按回车。

**Q: 安装完成后输入 hermes 提示"command not found"**
A: 关掉终端窗口，重新打开一个。如果还是找不到，重启电脑。

**Q: Windows 安装后中文显示乱码**
A: 在终端标题栏右键 → 属性 → 字体 → 选一个中文字体（如"新宋体"或"Consolas"）。

## 模型相关

**Q: 第一次用该选哪个模型？**
A: DeepSeek。免费，中文好，适合日常使用。去 platform.deepseek.com 注册拿 Key。

**Q: 模型 Key 是什么、怎么拿？**
A: API Key 是一串字母和数字混合的"通行证"。注册 DeepSeek 或 OpenRouter 后，在网站的"API Keys"或"密钥管理"页面创建。复制下来保存好。

**Q: DeepSeek 的 API Key 要钱吗？**
A: 注册就送免费额度。日常使用（搜东西、写东西、整文件）完全够用。如果用完了可以充值，价格很低。

**Q: 怎么切换模型？**
A: 在 Hermes 里说"帮我换一个模型"，或输入 `/model`。

**Q: 模型回复很慢怎么办？**
A: 可能原因：网络不好、模型服务器繁忙。试试：切到 Wi-Fi、换个模型、等几分钟再试。

## 使用相关

**Q: 它有时候胡说八道怎么办？**
A: AI 偶尔会"编造"信息，尤其是搜不到确切数据的时候。做法：重要信息要求它给出信息来源。如果你不确定它对不对，加一句"如果你不确定，就说不知道，不要编"。

**Q: 它能帮我操作微信/淘宝/其他 App 吗？**
A: 不能直接操作其他 App。它能操作的是你电脑上的文件、浏览器、终端。微信上它只是跟你聊天。

**Q: 它能看到我的私人文件吗？**
A: 只有你让它看的文件，它才会看。它不会自动扫描你的硬盘。它的搜索范围也限制在你告诉它的文件夹内。

**Q: 我输入的信息安全吗？**
A: 你的对话和文件发送给模型提供商（如 DeepSeek）处理。建议：涉及密码、银行卡号等敏感信息时，不要发给 AI。

**Q: 怎么关闭 Hermes？**
A: 在对话里输入 `/exit` 或 `/quit`，或者直接关终端。

**Q: 电脑关机后定时任务还会跑吗？**
A: 不会。Hermes 需要电脑开着才能运行。定时任务依托的是你的电脑。

## 费用相关

**Q: Hermes 本身收费吗？**
A: 不收费。Hermes 是开源软件，完全免费。

**Q: 那什么情况下会花钱？**
A: 花在"模型"上。你用的 AI 模型如果是付费的，会按使用量收费。但你可以一直用免费模型（DeepSeek 免费额度、OpenRouter 免费模型）。

**Q: 怎么控制成本？**
A: 三步：1. 用免费模型；2. 在 Hermes 设置里限制每次对话的最大使用量；3. 定期在模型提供商的网站查看用量。
