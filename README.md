## 这本书能帮你做什么

你是否遇到过这些场景：

- 搜了一堆网页，每个都要点开看，看完记不住
- 周报写了删、删了写，不知道怎么写才像样
- 几百个文件堆在一起，不知道哪个是哪个
- 想让 AI 自动帮你盯新闻、关注价格变化，但不知道怎么搞

这本书就是解决这些问题的。你不用学任何技术，会打字就能用。

每一章讲一个具体场景，跟着做，做完就能用。
