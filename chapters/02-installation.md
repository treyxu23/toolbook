## §02 装起来，5 分钟

### 02.1 你需要什么

三样东西：

- 一台电脑（Mac 或 Windows，都行）
- 能上网
- 一个叫"终端"的东西——别怕，它就是你电脑自带的一个黑框框

**预计时间**：5-10 分钟。运气好 5 分钟，遇到小问题的话 10 分钟。我已经走过一遍了，常见坑都标出来了，你跟着做就行。

### 02.2 打开终端

**Mac 用户**：按 `Command + 空格`，输入「终端」，回车。看到一个黑框框或白框框，那就是。

**Windows 用户**：按 `Win + R`，输入 `cmd`，回车。看到一个黑框框，那就是。

这个黑框框就是你和 Hermes 对话的地方。现在它还是空的，别怕。往下走。

### 02.3 安装

把下面这行字**完整复制**，贴到终端里，按回车：

```
curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash
```

> **怎么复制粘贴**：Mac 用 `Command + C` 复制、`Command + V` 粘贴。Windows 用 `Ctrl + C` 复制、在终端里右键粘贴。

然后等。屏幕上会跑一些字，1-2 分钟后停下来。

> **预期结果**：最后一行出现类似"Installation complete"或"安装完成"的字样。

> **注意**：如果提示需要密码，输入你电脑的开机密码（输入时屏幕不显示，正常，继续输完按回车）。

### 02.4 装不上怎么办

以下是我见过的最常见的 4 个问题。按顺序检查：

**问题 1：提示"curl: command not found"**

Mac 通常自带 curl。如果提示找不到：在终端输入 `xcode-select --install`，按提示安装完再试。Windows 需要先装 curl：去 https://curl.se/windows 下载安装。

**问题 2：Windows 提示"无法识别此命令"**

你可能不在 cmd 里。确认你打开的是"命令提示符"（cmd），不是 PowerShell。或者试试用 PowerShell 打开，然后输入同样的命令。

**问题 3：安装完输入 hermes，提示"command not found"**

关掉终端窗口，重新打开一个。如果还是找不到：重启电脑。还不行的话——Mac 用户在终端输入 `eval "$(/usr/libexec/path_helper)"` 然后重试。

**问题 4：Windows 安装过程中被杀毒软件拦截**

Windows Defender 或第三方杀毒软件可能会弹窗警告。选择"允许"或"信任"。如果不知道怎么操作，暂时关闭杀毒软件的实时保护，装完再打开。

> **如果上面 4 个都试了还是不行**：把错误信息复制下来，去 Hermes 的 GitHub 页面（github.com/NousResearch/hermes-agent）点 Issues，搜一下你的错误信息。大概率有人遇到过。

装完后，关掉终端，重新打开一个。然后输入：

```
hermes
```

回车。如果看到一段欢迎界面或一个对话提示，恭喜——装好了。

### 02.5 选一个模型

第一次用 Hermes，它会问你想用哪个模型。模型就是 AI 的"大脑"，不同模型能力不同、价格不同。

> **核心建议**：先用免费的，用顺手了再说付费的事。

我推荐两个免费方案（按推荐顺序）：

1. **DeepSeek** — 中文最好，完全免费。去 [platform.deepseek.com](https://platform.deepseek.com) 注册，拿到一个 API Key（一串字母数字混合），记下来。

2. **OpenRouter 免费模型** — 选择多，也有免费模型。去 [openrouter.ai](https://openrouter.ai) 注册拿 Key。

拿到 Key 后，在终端输入：

```
hermes setup model
```

按提示填入你的 Key，选对应的模型。不会的话，选第一个推荐选项就行。

> **我的经验**：我用 DeepSeek 跑了 3 个月。最长一次连续对话跑了将近两天，没崩过。中文理解能力比很多付费模型还好。对于日常用——搜东西、写文案、整文件——完全够用。

### 02.6 第一次对话

模型选好了。现在做第一件事。

在 Hermes 的对话界面里，输入：

> 帮我介绍一下你自己，一句话。

它应该回复你一段话。如果回复了，祝贺——你的 AI 助手已经就位。

接下来，输入一句更有用的：

> 帮我搜一下最近 3 天 AI 领域的 3 条重要新闻，每条用一句话总结。

它会真的打开浏览器去搜，然后把结果整理好给你。

看到这里，你已经完成了从零到第一次用 Hermes 的全过程。
